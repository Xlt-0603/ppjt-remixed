```
VillageScene.unity
├── Main Camera (正交, size=11.667, Physics2DRaycaster)
├── Canvas (Screen Space - Camera)
│   ├── ResourceBar (金币/宝石/科技)
│   ├── Toolbar (战斗/卡牌/商店/邮件)
│   ├── Btn_ScrollLeft / Btn_ScrollRight
│   ├── PanelRoot (隐藏, 各建筑面板)
│   └── EventSystem
└── VillageManagers
    ├── VillageManager (主控单例)
    ├── VillageDataManager (数据持久化)
    ├── BuildingManager (建筑管理器)
    ├── Ground (白色, sortingOrder=-1)
    └── Buildings (6栋建筑, SpriteRenderer + BoxCollider2D)
```

### 点击系统
- 点击检测在 `VillageManager.Update()` 中，`Input.GetMouseButtonUp` → 10px 拖拽判定 → `Physics2D.GetRayIntersection` → `VillageBuilding.NotifyClicked()`
- 放弃 `OnMouseDown`/`IPointerClickHandler`（团结引擎 1.8.5 下不可靠）
- 建筑被点击时：弹出对应 Panel + 缩放动画 (1.05x, 0.06s 放大 + 0.06s 恢复)

### 弹性边界
- 拖拽超出 `SetBounds(-49.5, 26.6)` 时阻力渐增 (`_dragResistance=0.3`)
- 松手后弹性回弹 (`_elasticStrength=5`)
- 左右滚动按钮也可超界后自动弹回
