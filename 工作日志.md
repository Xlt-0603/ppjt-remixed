## 8. 先锋冲锋（Vanguard）

### 数据配置（UnitData）
- 新增 `isVanguard`、`vanguardScanMin`、`vanguardScanMax`、`vanguardDamage`、`vanguardChargeBeats`

### 架构
- **组件化**：`VanguardController.cs` 独立组件，挂到 `UnitBase` 同级即可，不替换原脚本
- **时序**：`Awake` 订阅 `OnBeat`（回调先于 `UnitBase.Start` 注册），确保每拍先于单位处理冲锋逻辑

### 冲锋流程
1. **寻找目标**：每拍扫描前方 `vanguardScanMin`~`vanguardScanMax` 格内最近敌人（自动跳过攻击范围以内的）
2. **路径检测**：检查与目标之间的格子是否畅通（非塔单位即阻挡）
3. **蓄力**：`vanguardChargeBeats` 拍（默认 8 = 1 小节），期间设置 `SuppressNormalBehavior = true` 阻止移动/攻击
4. **执行冲锋**：
   - 非塔 + 后方格子空 → `ForceMove` 击退敌人 1 格 + 先锋冲到原位
   - 塔或后方被占 → 先锋冲到敌人前一格 + 只造成伤害
   - 冲锋后解禁 `SuppressNormalBehavior`

### 新增公共 API（UnitBase）
| 成员 | 说明 |
|------|------|
| `SuppressNormalBehavior` | 外部设为 true 时 `OnBeat` 跳过战斗/移动逻辑 |
| `Data` | 暴露 `data` 字段 |
| `ForceMove(GridPosition)` | 强制移动到指定格子（含 Leave/Occupy + 移动动画） |
| `SetGridPosition(GridPosition)` | 只改网格不带动画（已被 ForceMove 替代） |
