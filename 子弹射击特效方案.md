# 子弹射击特效方案

## 目标
用 Unity Particle System 自制子弹射击特效，不使用 LineRenderer。攻击时从射击者位置生成一个高速飞行的子弹粒子，命中目标位置后产生爆点。

## 方案一：Particle System 子弹束

### 构成
一个 Particle System 预制体，配合脚本控制发射时机和方向。

### 子弹束参数
- **Shape**: Cone（锥形），angle=0 → 沿直线射出
- **Start Speed**: 20–30（高速度，制造瞬间飞过的感觉）
- **Start Lifetime**: 根据攻击范围动态计算（`distance / speed`）
- **Start Size**: 0.1–0.2（小颗粒）
- **Start Color**: 攻击方颜色（玩家白/黄，敌人红/橙）
- **Max Particles**: 5–10（每次射击喷一簇）
- **Loop**: false
- **Prewarm**: false

### 发射脚本
在 `UnitBase` 或 `PierceController` / `VanguardController` 上新增 `_fireEffectPrefab` 字段，攻击时：

1. 在攻击者炮口位置 `Instantiate` 粒子预制体
2. 用 `ParticleSystem.EmitParams` 设置粒子的初始位置和方向
3. 让粒子朝向目标方向飞行
4. 在目标位置生成一个命中爆点（`FX_GunFlash` 或简单圆形粒子爆开）
5. 0.5s 后 `Destroy`

### 核心代码流程
```csharp
GameObject fx = Instantiate(_bulletEffectPrefab, firePos, Quaternion.identity);
ParticleSystem ps = fx.GetComponent<ParticleSystem>();

// 计算飞行距离和方向
float distance = Vector3.Distance(firePos, targetPos);
float speed = 25f;
ps.main.startLifetime = distance / speed;

// 面向目标
fx.transform.LookAt(targetPos);

// 命中爆点
// 可用 SimpleFX 的 FX_GunFlash 或自己起一个爆点粒子
```

## 方案二：单粒子手动控制（更精准）

不依赖 Particle System 的自动生命周期，而是用脚本逐帧更新一个粒子的位置：

```csharp
GameObject go = new GameObject("Bullet");
ParticleSystem ps = go.AddComponent<ParticleSystem>();
var emit = ps.Emit(new ParticleSystem.EmitParams(), 1);
ParticleSystem.Particle[] p = new ParticleSystem.Particle[1];
ps.GetParticles(p);

// 每帧更新 p[0].position
// 到达目标后在目标位置生成爆点
// kill 粒子并 Destroy(go)
```

方案二控制更精确，但实现略复杂，不适合多子弹同时飞行。**推荐方案一**。

## 爆点效果
命中目标后，在目标位置生成一个临时爆点粒子：
- 用 `ParticleSystem` + `Circle` Shape，向外爆散 5–10 个小点
- Start Size 0.05–0.15
- Start Lifetime 0.2–0.3s
- 白色/黄色，带一点随机颜色偏移
- 快速 fade out

## 用法
1. 在 Unity 里创建 `BulletEffect` 预制体（Particle System，按上述参数调）
2. 拖到单位预制体的 `_fireEffectPrefab` 字段
3. 攻击时自动调用

## 工作量预估
- 预制体调参：15min
- 脚本改动（UnitBase + PierceController）：10min
- 测试调优：15min
