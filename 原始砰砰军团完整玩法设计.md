## 一、游戏总览

### 1.1 游戏类型

像素风 **节奏卡牌策略对战** + **村庄模拟经营** 手游。

### 1.2 核心特色

- **节奏战斗**：所有豆丁的移动、攻击、被击飞均精准踩在 **4/4 拍**音乐鼓点上
- **横板推塔**：双方在 24 格直线战场上对战，摧毁对方哨塔获胜
- **村庄生活**：豆丁们在"砰砰村"中生活、互动、钓鱼、烹饪
- **公平竞技**：所有卡牌招募概率相同，承诺永久不重复

### 1.3 游戏背景

玩家作为"村长"，通过手机意外连入豆丁通讯网络，在砰砰大陆上指挥豆丁军团与其他玩家切磋，并与数据幽灵对抗。

豆丁是没有性别的像素生物，身体柔软，没有表情，靠 emoji 表达情绪，天生"踩点怪"。

## 二、主界面与村庄（砰砰村）

### 2.1 主界面结构

游戏主界面即为 **砰砰村** 的全景视图，村庄是豆丁们生活、训练的地方。界面布局如下：

```
┌──────────────────────────────────────────────┐
│  头像/等级 │  资源栏(金币/宝石/食玉/科技点)  │  ［设置］  │
├──────────────────────────────────────────────┤
│                                                │
│    ┌───┐  ┌───┐  ┌───┐  ┌───┐              │
│    │食堂│  │渔场│  │研究所│ │战斗中心│          │
│    └───┘  └───┘  └───┘  └───┘              │
│                                                │
│         （村庄场景，豆丁自由走动）              │
│                                                │
│    ┌───────────────┐                          │
│    │  可交互装饰物  │ ← 蹦床/椅子/大炮等      │
│    └───────────────┘                          │
│                                                │
├──────────────────────────────────────────────┤
│  ［卡牌编辑］  ［对战］  ［商店］  ［图鉴］    │
└──────────────────────────────────────────────┘
```

### 2.2 村庄功能建筑

| 建筑 | 解锁条件 | 功能 |
|------|----------|------|
| **战斗中心** | 默认 | 进入 PvP 匹配、PvE 冒险、2v2 双人模式 |
| **研究所** | 默认 | 消耗科技点解锁周边玩法（食堂、村庄建设等） |
| **食堂** | 研究所 8 级 | 料理制作、经营餐厅、获取豆丁碎片 |
| **渔场** | 村庄等级 | 钓鱼小游戏、收集鱼图鉴 |
| **村庄建设** | 研究所解锁 | 自由装扮村庄（前层/中层/背景层） |

### 2.3 村庄建设与装饰

- **解锁条件**：通过研究所使用科技点解锁
- **编辑模式**：长按建筑或点击右方 UI 进入
- **三层装饰**：前层 / 中层 / 背景层，自由搭配
- **装饰来源**：商店每天刷新出售
- **交互装饰**：部分装饰可与豆丁互动
  - **蹦床**：拖拽豆丁上去弹跳
  - **椅子**：豆丁坐下休息
  - **大炮**：豆丁被发射出去
  - **浴缸**：豆丁泡澡
- **好友互动**：可到好友村庄中白嫖装饰互动
- **豆丁管理**：通过左下角图鉴拖拽豆丁到村庄；拖回战斗中心则移除
- **豆丁对话**：村庄中的豆丁会随机触发有趣的对话气泡

### 2.4 村庄等级

- 村庄等级是村庄繁荣的指标
- 提高等级增加 **金币经营收益**
- 解锁更高级的 **建设方案**
- 建设方案增加 **收益上限** 和 **食堂菜单上限**
- 升级所需资源：金币、材料

### 2.5 科技点

- 通过 **提升村庄等级** 获取
- 在村中心的 **研究所** 消耗科技点解锁周边玩法

## 三、核心战斗系统

### 3.1 战斗概述

- 实时策略对抗，双方各 **8 张卡牌** 组成出战卡组
- 战场为 **24 格** 的直线水平网格
- 目标为 **摧毁对方哨塔**
- 所有豆丁的行动精准踩在 4/4 拍音乐节拍上

### 3.2 战场布局

```
我方哨塔  我方塔前  中场  敌方塔前  敌方哨塔
    │        │       │       │        │
   col0     col1-5  col6-17 col18-22  col23
```

区域划分：
- **我方塔下**（col 0-?）：部署区域
- **我方塔前**：进攻发起区域
- **敌方塔前**：敌方防守区域
- **敌方塔下**（col 23）：敌方哨塔位置

### 3.3 能量系统

- 每回合开始时获得能量：**阶段数 + 2**
- 消耗能量部署豆丁
- 能量会随回合推进逐渐增加

### 3.4 回合流程

```
每回合开始：
  1. 获得能量（阶段数 + 2）
  2. 玩家选择卡牌部署（消耗能量）
  3. 豆丁上场后自动前进与攻击
  4. 豆丁的移动/攻击都踩在4/4拍鼓点上
```

### 3.5 豆丁职业定位

| 职业 | 英文 | 定位 | 特点 |
|------|------|------|------|
| **先锋** | Vanguard | 前排近战 | 冲锋在前，近战输出 |
| **守卫** | Guard | 防御坦克 | 高血量，保护后排 |
| **射手** | Shooter | 远程输出 | 远程攻击，脆皮 |
| **斥候** | Scout | 刺客切入 | 高机动，切后排 |
| **法术** | Spell | 法术卡牌 | 特殊效果，范围伤害等 |

### 3.6 对战模式

| 模式 | 说明 |
|------|------|
| **天梯匹配** | 1v1 实时 PvP，赛季制段位 |
| **冒险关卡** | PvE 推图（如"迷途探寻"等） |
| **双人模式** | 2v2 合作对抗 |
| **娱乐模式** | 特殊规则玩法（如"飞跃乱斗"） |

### 3.7 主流战术流派

| 流派 | 核心思路 | 代表卡牌 |
|------|----------|----------|
| **自闭流** | 防守反击，蹭血取胜 | 镭射炮、唐吉诃德 |
| **敲门流** | 高伤害突击，快速破塔 | 巨斩、毁灭者 |
| **推进流** | 阵地推进，节奏控制 | 一刀斋、盖塔、机枪 |

## 四、卡牌收集与组卡

### 4.1 豆丁稀有度

- 豆丁只存在 **稀有度** 区别，没有绝对强度梯度
- 所有卡牌在卡池中 **招募概率相同**
- 承诺 **永久不重复**（不会抽到已有满级卡牌）

### 4.2 卡牌属性

```
卡牌属性（以温氏牛仔 5级为例）：
  ├─ 费用：2 费能量
  ├─ 品阶：金色
  ├─ 攻击：29
  ├─ 生命：117
  ├─ 攻击范围：1-6 格
  ├─ 定位：射手
  ├─ 攻击节奏：中等
  └─ 技能：阵地战（未移动时每层+1射程和20%攻击）
```

### 4.3 组卡规则

- 每套卡组 **8 张卡牌**
- 需要搭配不同职业位置
- 讲究 **阵容克制** 与 **上场时机**

### 4.4 皮肤系统

- 豆丁皮肤：在仓库 → 皮肤页面更换
- 哨塔皮肤：在卡牌编辑页面 → 哨塔按键更换
- 建筑皮肤：长按建筑 → 小房子图标更换

## 五、渔场玩法

### 5.1 概述

渔场是砰砰村的休闲小游戏，玩家在不同水域使用钓具钓鱼，收集鱼图鉴。钓鱼获得的鱼可用于食堂烹饪或完成成就。

### 5.2 渔场区域

| 区域名称 | 环境描述 | 可钓鱼类 |
|----------|----------|----------|
| **林间小溪** | 森林中的清澈溪流 | 淡水小型鱼 |
| **入海口** | 河流入海交界处 | 咸淡水混合鱼种 |
| **林中密潭** | 森林深处的隐秘水潭 | 稀有淡水鱼 |
| **寒流之下** | 寒冷水域 | 冷水鱼、深水鱼 |
| **平行海域** | 特殊的平行空间海域 | 珍稀/霸主级鱼类 |

### 5.3 钓鱼装备

#### 鱼竿（鱼竿）
- 决定可钓鱼类的 **长度区间**
- 不同鱼竿适配不同体型鱼类
- 类型：海钓杆、溪流竿等

#### 鱼钩（鱼钩）
- 影响钓鱼的 **特质匹配**
- 如：爆炸钩等

#### 鱼饵（鱼饵）
- 影响钓鱼的 **特质匹配**
- 如：活饵、面团饵等

### 5.4 鱼的属性

```
鱼类属性：
  ├─ 名称：如小王乌贼、深赤眉、火焰泥鳅等
  ├─ 体型大小：
  │   ├─ 小型鱼：1 格长度
  │   ├─ 中型鱼：2 格长度
  │   ├─ 大型鱼：3 格长度
  │   └─ 霸主：5 格长度
  ├─ 特质标签：如凶猛、热带、寒带等
  ├─ 所在区域：哪个渔场可钓
  └─ 鱼类图鉴：金冠（最长）/ 银冠（最短）
```

### 5.5 钓鱼机制

1. **长度匹配**：鱼竿长度区间必须与鱼的长度区间有重叠，才能钓上对应鱼
2. **特质匹配**：鱼竿/鱼钩/鱼饵的特质与目标鱼匹配时，概率大幅提升
3. **体型影响**：体型越大的鱼，钓出概率越低（大型鱼 < 中型鱼 < 小型鱼）
4. **金冠/银冠**：
   - 金冠 = 该鱼种的最大长度个体
   - 银冠 = 该鱼种的最小长度个体
   - 需要更精准的装备匹配才能钓到

### 5.6 钓鱼效率技巧

- 高概率是建立在 **基数足够大** 的前提下（如万分之一到万分之二）
- 使用对应特质的鱼竿 + 鱼钩 + 鱼饵组合，可最大化概率
- 示例：想钓金冠小王乌贼 → 海钓杆 + 爆炸钩 + 活饵（满足长度区间 + 凶猛特质 + 大型鱼）

### 5.7 钓鱼成就

- 完成鱼类图鉴收集可获得成就奖励
- 不同区域有独立的图鉴完成度

## 六、食堂玩法

### 6.1 概述

食堂是砰砰村的核心休闲经营玩法，玩家需要研发菜谱、制作料理、服务顾客，获取金币和豆丁碎片等奖励。

### 6.2 解锁条件

- **研究所等级达到 8 级** 解锁

### 6.3 核心流程

```
顾客下单 → 选择菜谱 → 分配厨师(豆丁) → 制作料理 → 上菜 → 获取收益
```

### 6.4 菜品等级体系

| 等级 | 说明 |
|------|------|
| **SS** | 最高等级，最稀有 |
| **S** | 高级，高收益 |
| **A** | 优良 |
| **B** | 良好 |
| **C** | 普通 |
| **D** | 基础 |

菜品等级由 **味道** 和 **价值** 两个维度综合决定。

### 6.5 食材系统

- 食材分为 **基础食材** 和 **高级食材**
- 基础食材：可在商人处无限量购买（面粉、蛋、盐、菠菜、番茄等）
- 高级食材：需完成任务获取，或使用宝石购买
- 部分食材为 **季节限定** 或 **活动限定**

### 6.6 厨师系统（豆丁）

- 豆丁拥有 **厨力等级**（做菜等级），初始 1 级，最高可升
- 不同豆丁有不同 **技能/特长**，影响料理品质
- 豆丁的厨力等级必须 **大于等于食谱难度**
- 推荐厨力等级：**5 级及以上** 可稳定做出 S 级菜品
- 示例厨师搭配：剑士、锋卫、弓箭手、击剑手、狂战士等

### 6.7 菜谱研发

- 玩家需要自行探索或参考配方组合食材
- 菜谱由 **食材组合 + 厨师搭配** 决定
- 同一菜谱可由不同食材组合达成
- 研发成功后可反复使用该菜谱

### 6.8 经营收益机制

```
经营收益 = 完成奖励 + 小费

单食谱小费 = 食客基础小费 + 0.1 × 食谱价值
单食谱小费上限 = 食客基础小费 + 评价点赞数
```

- 当食谱价值大于阈值，且能让该偏好所有食客都五星好评时，收益达到上限
- 受招待技能触发概率、招待食客不同等因素影响，实际收益有浮动（约 ±400）

### 6.9 食堂阶段

| 阶段名称 | 说明 |
|----------|------|
| **试营业** | 初始阶段，简单菜品 |
| **街角小店** | 升级解锁，更多菜谱 |
| **鲜炒大排档** | 中级阶段，收益提升 |
| **宝石餐厅** | 最高阶段，极限收益 |

### 6.10 S 级菜谱示例

| 菜名 | 食材 | 推荐厨师 |
|------|------|----------|
| 沙拉 | 菠菜+甜果+白盐 | 剑士、锋卫、弓箭手 |
| 水果派 | 白面粉+甜果+蛋 | 击剑手、狂战士、狙击手 |
| 花蛤汤 | 花蛤+白盐 | 飞斧将、锋卫、幻术师 |
| 糖醋肉 | 蛋+牛排+番茄+柠檬+红菜头 | 锤、平村大圣、牛仔系列 |
| 番茄蛋汤 | 蛋+番茄+五花+菠菜 | 枪士、锋卫、双枪 |
| 凉拌菜 | 番茄+菠菜+五花+蛋 | 飞斧将+平村大圣+弓箭手 |
| 土豆泥 | 土豆+鸡腿+鸡腿+面粉 | 飞斧将、狂战士、牛仔 |
| 焦糖布丁 | 蛋+牛奶+白糖 | 飞斧将、滑板小子、镭射炮 |
| 黑曜石蛋糕 | （高级食材） | 击剑手+镭射炮+小恶魔（15级+） |
| 水晶牛肉沙拉 | 牛肉（需发酵香调） | 拳击手+狙击手+白忍+温氏牛仔+狼仔 |
| 海鲜拼盘 | 海鲜（限定食谱） | 锋卫+小恶魔 |
| 坛四宝 | 需热带香调 | 小贩+弓箭手+社交达人+小方盾+科学法师+爱吃仔 |
| 麻辣香锅 | 需蘑菇+辛辣香调 | 小木匠+小贩+小斧子+队长+钢膛 |

### 6.11 顾客好感度

- 满足顾客偏好 → 好感度提升
- 好感度提升 → 解锁更多顾客类型/奖励
- 通用好感度食谱：使用 **4 格食谱** 满足前期 99% 需求
- 核心策略：用基础食材做出尽量高评分的菜品，快速刷满好感度

### 6.12 食客偏好示例

| 食客 | 偏好类型 |
|------|----------|
| 拳击手 | 特定菜品 |
| 大哥 | 特定菜品 |
| 转换者 | 特定菜品 |
| 狙击手 | 特定菜品 |
| 社交达人 | 果香/花香 |
| 小方盾 | 牛里脊+洋葱 |

## 七、其他系统

### 7.1 冒险模式（PVE）

- 提供 PvE 挑战关卡
- 通关获取资源奖励
- 如"迷途探寻"等主题活动关卡
- 包含特殊 BOSS 战和解谜关卡（数据谜题 1~11 关）

### 7.2 双人模式（2v2）

- 与好友组队 2v2 对战
- 合作策略配合

### 7.3 娱乐模式

- 特殊规则玩法
- 如"飞跃乱斗"等限时活动

### 7.4 赛季系统

- **天梯赛季**：定期更新赛季（S7、S8 等）
- 赛季奖励：皮肤、宝石、荣誉点等
- 赛季通行证/战令（含皮肤奖励）
- 赛季扭蛋皮肤

### 7.5 兑换码系统

- 定期发布礼包码
- 兑换码有时效限制
- 同一账号每个兑换码仅限使用一次

### 7.6 联动活动

- 与其他 IP 联动（如《不思议迷宫》）
- 限时活动任务
- 联动限定奖励

### 7.7 皮肤系统

- 豆丁皮肤：仓库 → 皮肤更换
- 哨塔皮肤：卡牌编辑页面更换
- 建筑皮肤：长按建筑更换
- 来源：赛季战令、扭蛋、活动奖励

## 八、经济系统与资源

### 8.1 资源类型

| 资源 | 用途 |
|------|------|
| **金币** | 基础货币，升级建筑、购买基础食材 |
