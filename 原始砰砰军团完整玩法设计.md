## 一、游戏总览

### 1.1 游戏类型

像素风 **节奏卡牌策略对战** + **村庄模拟经营** 手游。

### 1.2 核心特色

- **节奏战斗**：所有豆丁的移动、攻击、被击飞均精准踩在 **4/4 拍**音乐鼓点上
- **横板推塔**：双方在 24 格直线战场上对战，摧毁对方哨塔获胜
- **村庄生活**：豆丁们在"砰砰村"中生活、互动、钓鱼、烹饪
- **公平竞技**：所有卡牌招募概率相同，承诺永久不重复

### 1.3 游戏背景

玩家作为"村长"，通过手机意外连入豆丁通讯网络，在砰砰大陆上指挥豆丁军团与其他玩家切磋，并与数据幽灵对抗。

豆丁是没有性别的像素生物，身体柔软，没有表情，靠 emoji 表达情绪，天生"踩点怪"。

## 二、主界面与村庄（砰砰村）

### 2.1 主界面结构

游戏主界面即为 **砰砰村** 的全景视图，村庄是豆丁们生活、训练的地方。界面布局如下：

```
┌──────────────────────────────────────────────┐
│  头像/等级 │  资源栏(金币/宝石/食玉/科技点)  │  ［设置］  │
├──────────────────────────────────────────────┤
│                                                │
│    ┌───┐  ┌───┐  ┌───┐  ┌───┐              │
│    │食堂│  │渔场│  │研究所│ │战斗中心│          │
│    └───┘  └───┘  └───┘  └───┘              │
│                                                │
│         （村庄场景，豆丁自由走动）              │
│                                                │
│    ┌───────────────┐                          │
│    │  可交互装饰物  │ ← 蹦床/椅子/大炮等      │
│    └───────────────┘                          │
│                                                │
├──────────────────────────────────────────────┤
│  ［卡牌编辑］  ［对战］  ［商店］  ［图鉴］    │
└──────────────────────────────────────────────┘
```

### 2.2 村庄功能建筑

| 建筑 | 解锁条件 | 功能 |
|------|----------|------|
| **战斗中心** | 默认 | 进入 PvP 匹配、PvE 冒险、2v2 双人模式 |
| **研究所** | 默认 | 消耗科技点解锁周边玩法（食堂、村庄建设等） |
| **食堂** | 研究所 8 级 | 料理制作、经营餐厅、获取豆丁碎片 |
| **渔场** | 村庄等级 | 钓鱼小游戏、收集鱼图鉴 |
| **村庄建设** | 研究所解锁 | 自由装扮村庄（前层/中层/背景层） |

### 2.3 村庄建设与装饰

- **解锁条件**：通过研究所使用科技点解锁
- **编辑模式**：长按建筑或点击右方 UI 进入
- **三层装饰**：前层 / 中层 / 背景层，自由搭配
- **装饰来源**：商店每天刷新出售
- **交互装饰**：部分装饰可与豆丁互动
  - **蹦床**：拖拽豆丁上去弹跳
  - **椅子**：豆丁坐下休息
  - **大炮**：豆丁被发射出去
  - **浴缸**：豆丁泡澡
- **好友互动**：可到好友村庄中白嫖装饰互动
- **豆丁管理**：通过左下角图鉴拖拽豆丁到村庄；拖回战斗中心则移除
- **豆丁对话**：村庄中的豆丁会随机触发有趣的对话气泡

### 2.4 村庄等级

- 村庄等级是村庄繁荣的指标
- 提高等级增加 **金币经营收益**
- 解锁更高级的 **建设方案**
- 建设方案增加 **收益上限** 和 **食堂菜单上限**
- 升级所需资源：金币、材料

### 2.5 科技点

- 通过 **提升村庄等级** 获取
- 在村中心的 **研究所** 消耗科技点解锁周边玩法

## 三、核心战斗系统

### 3.1 战斗概述

- 实时策略对抗，双方各 **8 张卡牌** 组成出战卡组
- 战场为 **24 格** 的直线水平网格
- 目标为 **摧毁对方哨塔**
- 所有豆丁的行动精准踩在 4/4 拍音乐节拍上

### 3.2 战场布局

```
我方哨塔  我方塔前  中场  敌方塔前  敌方哨塔
    │        │       │       │        │
   col0     col1-5  col6-17 col18-22  col23
```

区域划分：
- **我方塔下**（col 0-?）：部署区域
- **我方塔前**：进攻发起区域
- **敌方塔前**：敌方防守区域
- **敌方塔下**（col 23）：敌方哨塔位置

### 3.3 能量系统

- 每回合开始时获得能量：**阶段数 + 2**
- 消耗能量部署豆丁
- 能量会随回合推进逐渐增加

### 3.4 回合流程

```
每回合开始：
  1. 获得能量（阶段数 + 2）
  2. 玩家选择卡牌部署（消耗能量）
  3. 豆丁上场后自动前进与攻击
  4. 豆丁的移动/攻击都踩在4/4拍鼓点上
```

### 3.5 豆丁职业定位

| 职业 | 英文 | 定位 | 特点 |
|------|------|------|------|
| **先锋** | Vanguard | 前排近战 | 冲锋在前，近战输出 |
| **守卫** | Guard | 防御坦克 | 高血量，保护后排 |
| **射手** | Shooter | 远程输出 | 远程攻击，脆皮 |
| **斥候** | Scout | 刺客切入 | 高机动，切后排 |
| **法术** | Spell | 法术卡牌 | 特殊效果，范围伤害等 |

### 3.6 对战模式

| 模式 | 说明 |
|------|------|
| **天梯匹配** | 1v1 实时 PvP，赛季制段位 |
| **冒险关卡** | PvE 推图（如"迷途探寻"等） |
| **双人模式** | 2v2 合作对抗 |
| **娱乐模式** | 特殊规则玩法（如"飞跃乱斗"） |

### 3.7 主流战术流派

| 流派 | 核心思路 | 代表卡牌 |
|------|----------|----------|
| **自闭流** | 防守反击，蹭血取胜 | 镭射炮、唐吉诃德 |
| **敲门流** | 高伤害突击，快速破塔 | 巨斩、毁灭者 |
| **推进流** | 阵地推进，节奏控制 | 一刀斋、盖塔、机枪 |

## 四、卡牌收集与组卡

### 4.1 豆丁稀有度

- 豆丁只存在 **稀有度** 区别，没有绝对强度梯度
- 所有卡牌在卡池中 **招募概率相同**
- 承诺 **永久不重复**（不会抽到已有满级卡牌）

### 4.2 卡牌属性

```
卡牌属性（以温氏牛仔 5级为例）：
  ├─ 费用：2 费能量
  ├─ 品阶：金色
  ├─ 攻击：29
  ├─ 生命：117
  ├─ 攻击范围：1-6 格
  ├─ 定位：射手
  ├─ 攻击节奏：中等
  └─ 技能：阵地战（未移动时每层+1射程和20%攻击）
```

### 4.3 组卡规则

- 每套卡组 **8 张卡牌**
- 需要搭配不同职业位置
- 讲究 **阵容克制** 与 **上场时机**

### 4.4 皮肤系统

- 豆丁皮肤：在仓库 → 皮肤页面更换
- 哨塔皮肤：在卡牌编辑页面 → 哨塔按键更换
- 建筑皮肤：长按建筑 → 小房子图标更换

## 五、渔场玩法

### 5.1 概述

渔场是砰砰村的休闲小游戏，玩家在不同水域使用钓具钓鱼，收集鱼图鉴。钓鱼获得的鱼可用于食堂烹饪或完成成就。

### 5.2 开启条件

- 玩家等级达到要求后，在村中心 → 建设方案 → 建造渔场
- 渔场占地 7 格地面
- 前提：完成"开荒工具"和"村庄编辑"后解锁

### 5.3 渔场区域

共 5 个钓鱼区域，通过累计垂钓次数逐步解锁：

| 区域名称 | 环境描述 | 解锁条件 | 可钓鱼类 |
|----------|----------|----------|----------|
| **林间小溪** | 森林中的清澈溪流 | 默认开启 | 淡水小型鱼 |
| **入海口** | 河流入海交界处 | 累计垂钓 300 次 | 咸淡水混合鱼种 |
| **林中密潭** | 森林深处的隐秘水潭 | 村中心新增钓场 | 稀有淡水鱼 |
| **寒流之下** | 寒冷水域 | 累计垂钓次数 | 冷水鱼、深水鱼 |
| **平行海域** | 特殊平行空间海域 | 累计垂钓次数 | 珍稀/霸主级鱼类 |

### 5.4 核心钓鱼机制

#### 5.4.1 两种模式

**手动垂钓（节奏音游）**：
1. 进入钓鱼界面，进度条上的光标左右飘动
2. 光标飘到钓钩图标区域时点击
3. **Perfect**：精准点击图标中心 → 进度条大幅增长
4. **Good**：点到图标边缘 → 进度条小幅增长
5. 点到红色区域（图标外）→ 无增长
6. 进度条集满 → 成功钓上一条鱼
7. 需要持续操作，手感与"打节拍"类似

**自动垂钓（挂机模式）**：
1. 一键开启自动钓鱼
2. 进度条自动缓慢增长
3. 可随时点击"收竿"结束
4. 挂机效率由装备（鱼箱容量、小猫速度）决定
5. 推荐日常使用，省心省力

#### 5.4.2 核心概率公式

```
钓鱼成功判定（逐层过滤）：
  第 1 层（硬条件）：鱼竿长度区间 ∩ 鱼长度区间 ≠ ∅
    └─ 不重叠 → 该鱼不可能钓到
  第 2 层（基础概率）：基于体型
    ├─ 小型鱼（1 格）→ 基础概率最高
    ├─ 中型鱼（2 格）→ 基础概率中等
    ├─ 大型鱼（3 格）→ 基础概率低
    └─ 霸主（5 格）→ 基础概率极低
  第 3 层（特征修正）：鱼竿/鱼钩/鱼饵特质匹配
    ├─ 0 项匹配 → 基础概率
    ├─ 1 项匹配 → 概率 ×1.5~2
    ├─ 2 项匹配 → 概率 ×3~4
    └─ 3 项全匹配 → 概率最大化（如万分之一→万分之二）
  第 4 层（金冠/银冠修正）：
    ├─ 金冠 / 银冠概率远低于普通个体
    └─ 需要更精准的装备匹配才能触及
```

> 注意：所谓"高概率"是基数足够大前提下的统计学改善（如万分之一到万分之二），钓几千次才出稀有鱼是常态。

#### 5.4.3 鱼的属性结构

```
鱼类属性：
  ├─ 名称：如小王乌贼、深赤眉、火焰泥鳅等
  ├─ 体型大小（决定长度区间）：
  │   ├─ 小型鱼：1 格长度（如深赤眉 → 1 格）
  │   ├─ 中型鱼：2 格长度（如火焰泥鳅 → 2 格）
  │   ├─ 大型鱼：3 格长度（如巨赤眉 → 3 格）
  │   └─ 霸主：5 格长度（如利维坦 → 5 格）
  ├─ 特质标签：凶猛、热带、寒带、灵体、稀有等
  ├─ 所在区域：哪个渔场可钓
  ├─ 长度变体：同种鱼有长度范围（如 95~200mm）
  │   ├─ 金冠：该鱼种最大长度个体
  │   └─ 银冠：该鱼种最小长度个体
  └─ 尺寸波动：每位玩家图鉴记录不同，存在个体差异
```

### 5.5 钓鱼装备

#### 5.5.1 鱼竿（核心 — 决定长度区间）

| 鱼竿 | 覆盖尺寸 | 满级长度区间 | 性价比 | 获取建议 |
|------|----------|-------------|--------|---------|
| 普通竿 | 小型鱼 | 初始赠送 | 前期过渡 | 适度升级减少空竿率即可 |
| **迷你竿** | 小型鱼 | 专精小型 | ★★★ | 推荐解锁 |
| **达人竿** | 综合 | 80~250mm | ★★★★★ | **前期核心**，18 宝石优先入手 |
| **海钓竿** | 大型/霸主 | 3~17 格 | ★★★★ | 覆盖绝大部分鱼种 |
| **利维坦竿** | 霸主专用 | 唯一尺寸专用 | ★★★ | 后期图鉴冲刺必备 |
| 大师竿（海域限定） | 105~235mm | 5% 失误率 | ★★★ | 海域后期，非必备 |

> **前期建议**：优先解锁达人竿 → 迷你竿 + 海钓竿 + 利维坦竿（四竿满级可覆盖所有鱼尺寸）

#### 5.5.2 鱼钩（特质匹配）

鱼钩带有特定倾向，提升对应特质鱼的捕获率：

| 鱼钩类型 | 擅长对应 |
|----------|----------|
| 普通钩 | 无倾向 |
| 爆炸钩 → 擅长凶猛类 | 凶猛特质鱼 |
| 灵体钩 → 擅长灵体类 | 灵体特质鱼 |
| 冷水钩 → 擅长冷水类 | 寒带/冷水特质鱼 |
| 稀有钩 → 擅长稀有/金冠/银冠 | 稀有/金冠/银冠个体 |

- 不同鱼钩对应不同类型，收集图鉴时针对性切换
- 金冠/银冠需要对应鱼钩大幅提升概率

#### 5.5.3 鱼饵（体型匹配）

| 鱼饵类型 | 效果 |
|----------|------|
| 普通鱼饵 | 无加成 |
| **窝料** | 减少 3% 失误率（推荐常驻） |
| 1 格饵 | 专精小型鱼（1 格体型） |
| 2 格饵 | 专精中型鱼（2 格体型） |
| 3 格饵 | 专精大型鱼（3 格体型） |

- 钓特定体型鱼时，使用对应格数的鱼饵
- 霸主（5 格）不在鱼饵专精范围内，靠鱼竿覆盖

#### 5.5.4 辅助设备

| 设备 | 功能 | 满级效果 | 升级建议 |
|------|------|----------|---------|
| **鱼箱** | 提升自动垂钓容量 | 自动钓 80 条 | ★ 优先满级 |
| **小猫** | 提升钓鱼速度 | 速度最大化 | ★ 优先满级 |

> 前期先将鱼箱和小猫升满，可大幅提升挂机效率。

### 5.6 钓鱼产出

| 产出 | 用途 |
|------|------|
| **鱼币** | 购买/升级鱼竿、鱼钩、鱼饵、鱼箱、小猫 |
| **宝石** | 钓鱼概率产出，高级货币 |
| **鱼类** | 送至食堂制作料理、完成图鉴收集 |
| **成就奖励** | 各区域图鉴完成度达标后领取 |

### 5.7 策略指引

#### 新手起步
1. 先升满鱼箱 + 小猫 → 自动挂机效率最大化
2. 积攒鱼币 → 解锁达人竿（18 宝石）
3. 达人竿 + 窝料 → 日常自动钓鱼积累
4. 逐步解锁迷你竿、海钓竿、利维坦竿

#### 图鉴收集
1. 先确认目标鱼的长度区间 → 选择对应鱼竿
2. 查看目标鱼的特质 → 切换对应鱼钩
3. 确认目标鱼体型（几格）→ 使用对应鱼饵
4. 三件套匹配 → 概率最大化
5. 金冠/银冠冲刺：在上述基础上使用稀有钩

#### 示例
```
目标：金冠小王乌贼（入海口/平行海域）
  ├─ 长度区间：大型鱼 → 海钓竿（3~17 格）
  ├─ 特质：凶猛 → 爆炸钩
  ├─ 体型：3 格 → 3 格饵
  └─ 金冠 → 稀有钩替代爆炸钩（可选）
  期望概率：基础万分之几 → 装备匹配后翻倍
```

### 5.8 钓鱼成就

- 每个区域有独立的图鉴完成度
- 完成全区域图鉴收集可获得终极成就
- 成就奖励：宝石、限定称号等

## 六、食堂玩法

### 6.1 概述

食堂是砰砰村的核心休闲经营玩法，玩家需要研发菜谱、制作料理、服务顾客，获取金币和豆丁碎片等奖励。

### 6.2 解锁条件

- **研究所等级达到 8 级** 解锁

### 6.3 核心流程

```
顾客下单 → 选择菜谱 → 分配厨师(豆丁) → 制作料理 → 上菜 → 获取收益
```

### 6.4 菜品等级体系

| 等级 | 说明 |
|------|------|
| **SS** | 最高等级，最稀有 |
| **S** | 高级，高收益 |
| **A** | 优良 |
| **B** | 良好 |
| **C** | 普通 |
| **D** | 基础 |

菜品等级由 **味道** 和 **价值** 两个维度综合决定。

### 6.5 食材系统

- 食材分为 **基础食材** 和 **高级食材**
- 基础食材：可在商人处无限量购买（面粉、蛋、盐、菠菜、番茄等）
- 高级食材：需完成任务获取，或使用宝石购买
- 部分食材为 **季节限定** 或 **活动限定**

### 6.6 厨师系统（豆丁）

- 豆丁拥有 **厨力等级**（做菜等级），初始 1 级，最高可升
- 不同豆丁有不同 **技能/特长**，影响料理品质
- 豆丁的厨力等级必须 **大于等于食谱难度**
- 推荐厨力等级：**5 级及以上** 可稳定做出 S 级菜品
- 示例厨师搭配：剑士、锋卫、弓箭手、击剑手、狂战士等

### 6.7 菜谱研发

- 玩家需要自行探索或参考配方组合食材
- 菜谱由 **食材组合 + 厨师搭配** 决定
- 同一菜谱可由不同食材组合达成
- 研发成功后可反复使用该菜谱

### 6.8 经营收益机制

```
经营收益 = 完成奖励 + 小费

单食谱小费 = 食客基础小费 + 0.1 × 食谱价值
单食谱小费上限 = 食客基础小费 + 评价点赞数
```

- 当食谱价值大于阈值，且能让该偏好所有食客都五星好评时，收益达到上限
- 受招待技能触发概率、招待食客不同等因素影响，实际收益有浮动（约 ±400）

### 6.9 食堂阶段

| 阶段名称 | 说明 |
|----------|------|
| **试营业** | 初始阶段，简单菜品 |
| **街角小店** | 升级解锁，更多菜谱 |
| **鲜炒大排档** | 中级阶段，收益提升 |
| **宝石餐厅** | 最高阶段，极限收益 |

### 6.11 顾客好感度

- 通用好感度食谱：使用 **4 格食谱** 满足前期 99% 需求
- 核心策略：用基础食材做出尽量高评分的菜品，快速刷满好感度

- 如"迷途探寻"等主题活动关卡

```
  ├─ 食堂顾客打赏

豆丁碎片：
  ├─ 食堂经营（核心产出途径）
  ├─ 卡牌招募
  └─ 活动奖励
```
